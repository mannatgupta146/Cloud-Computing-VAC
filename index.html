Cloud Computing Quiz Answers

Module quiz: Computer Networking

  1. WAN (A Wide Area Network covers a larger geographical area than a MAN, such as countries or continents).
  2. A mesh network has multiple paths (This is its defining feature, providing redundancy).
  3. True (The bus topology became obsolete as more efficient and reliable star topologies, which use hardware like hubs, switches, and routers, became the standard).
  4. Security risks (Transmitting data over radio waves (Wi-Fi) is inherently less secure than a wired connection).
  5. hexadecimal numbers (IPv4 uses dotted-decimal notation, while IPv6 uses hexadecimal notation).
  6. Network and host sections of an IP address (The subnet mask is used to divide an IP address into these two parts).
  7. IPv6 has built in security (IPsec is a mandatory component of IPv6) and IPv6 does not support Broadcast traffic (It uses multicast and anycast instead).
  8. C-class (In classful addressing, Class C networks were designed for small networks, e.g., the 192.168.1.0 range).
  9. True (The default gateway is the IP address of the router that a computer uses to send data to any other network).
  10. False (The default subnet mask for a Class B network is 255.255.0.0. The mask 255.0.0.0 is for a Class A network).

Module quiz: Network devices and protocols

  1. False (A router is used to join different networks and subnets. A switch connects devices within the same network).
  2. Presentation, Application, and Session (The TCP/IP Application layer combines the functions of these top three layers of the OSI model).
  3. Hybrid cloud (This is the specific term for a setup that combines a private cloud with public cloud services).
  4. Remote (A "Remote Access VPN" allows individual users, like someone at home, to connect to the office network).
  5. Email (SMTP is for sending email, while IMAP and POP3 are for receiving email).
  6. TLS and SSL (HTTPS is HTTP secured by either Transport Layer Security (TLS) or its predecessor, Secure Sockets Layer (SSL)).
  7. 443 (HTTP uses port 80, and the secure version, HTTPS, uses port 443).
  8. DNS amplification (This is a DDoS attack that uses open DNS servers to flood a target with a large volume of DNS response traffic).
  9. Quality of Service (QOS) (QoS is the suite of tools and mechanisms used to manage network traffic).
  10. grep, findstr, and Select-String (grep is for Linux/macOS, findstr is for Windows Command Prompt, and Select-String is for Windows PowerShell).

Module quiz: Network security

  1. Request-response model (This is the most common model, where a client sends a request and the server sends a response).
  2. Rules (Firewall rules, also known as access control lists or ACLs, use these components to decide whether to allow or block traffic).
  3. Packet-filtering (This is the most basic type of firewall, which operates at the network layer).
  4. Man-in-the-middle (This attack describes an attacker secretly positioning themselves between two communicating parties).
  5. Penetration testing (Specifically, this is a phishing simulation, which is a common part of a penetration test).
  6. Listening to the communication between two network devices (Network sniffing involves capturing and inspecting data packets).
  7. Administrator access and Superuser access (Both terms refer to accounts with the highest level of privileges).
  8. False (RADIUS, TACACS, and LDAP are primarily authentication protocols, which verify who you are).
  9. True (This four-step process is often remembered by the acronym DORA: Discover, Offer, Request, Acknowledge).
  10. DHCP relay (A DHCP relay agent forwards DHCP broadcast packets between different subnets).

Module quiz: Cloud computing and networking

  1. Worldwide access, Scalability, and Easy to manage (Cloud platforms like Azure are designed to be globally accessible, scalable, and managed centrally).
  2. Cost and Dependence on Internet connectivity (At a very large scale, costs can be a disadvantage, and a reliable internet connection is a critical dependency).
  3. Hosted (Type 1 hypervisors are "bare metal," and Type 2 hypervisors are "hosted" on top of an OS).
  4. Compute-optimized (Real-time big data processing requires high-performance processors for computations).
  5. Machine learning (ML) (ML is the specific field of AI focused on algorithms that learn from data to identify patterns).
  6. (a) Reactive machines, (b) General AI (Reactive machines are the simplest AI. Artificial General AI (AGI) is the theoretical, future type of AI).
  7. OpEx (Operational Expenditure (OpEx) is a pay-as-you-go model for ongoing costs, like subscriptions).
  8. SaaS (Software as a Service. Microsoft 365 is a complete software application accessed over the internet).
  9. True (In the SaaS model, the provider manages everything; the customer just uses the software).
  10. True (SaaS applications are delivered over the internet, so bandwidth is critical for them to function).
